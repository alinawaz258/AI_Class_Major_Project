What are Q-Values?
Every node on the map has a Q-value — a number the AI learned meaning "how good is it to be here?"

At the start, all Q-values are 0. After each step, the Bellman equation updates the value of that node.

Enable "Q-Values" overlay after training — brighter blue = AI learned this location is valuable on the way to the goal.
What is the Heatmap?
Shows where the agent explored during training — everywhere it wandered while learning.

Early = spread everywhere (random). End = concentrated along optimal corridor.

Enable "Heatmap" overlay — orange density circles, thick clusters = heavily explored zones.
What is the Policy?
The agent's final decision map — at every node, an arrow showing the best direction.

Nobody programmed these arrows. The agent built this entire map from scratch through trial and error.

Enable "Policy" overlay — purple arrows forming a flow field across the city.
